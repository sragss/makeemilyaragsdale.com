# MakeEmilyARagsdale.com — Project Context

Wedding website for Emily Devery & Sam Ragsdale. February 27, 2027. San Miguel de Allende, Mexico.

## Architecture

- **Next.js 16** App Router with Turbopack. All pages use Server Components by default.
- **Tailwind CSS v4** + **shadcn/ui** (new-york style, Radix primitives). Components in `src/components/ui/`.
- **Neon Postgres** via `@neondatabase/serverless` + **Drizzle ORM**. Schema in `src/db/schema.ts`, lazy DB init in `src/db/index.ts`.
- **Framer Motion** for animations (RSVP toggle, form transitions, botanical confetti).
- **NoiseBackground** component (`src/components/ui/noise-background.tsx`) — animated gradient + noise texture used for form inputs and toggle controls.

## Deployment

- **Vercel project**: `sragss-projects/makeemilyaragsdale-com`
- **GitHub**: `sragss/makeemilyaragsdale.com` — push to `main` auto-deploys to production

To deploy: just `git push origin main`. Vercel handles the rest.

## Database

- **Neon project**: `makeemilyaragsdale` (org: Sam, region: aws-us-east-2)
- **Schema changes**: Edit `src/db/schema.ts`, then `pnpm db:push`
- **Seed data**: `pnpm db:seed` — creates 3 test invites with random pronounceable codes
- **Tables**: `invites` (code, hotel_eligible, max_guests, notes, address), `guests` (name, attending, email, phone, dietary, plus_one), `hotel_bookings` (will_book, booking_complete, booking_value)

## RSVP Flow

1. Guest visits `/rsvp` or `/rsvp/EBR-XXXXX` (direct link)
2. Enters pronounceable code (generated by `src/lib/codes.ts` — CVCVC pattern like TOVIK, BALEN)
3. URL updates to `/rsvp/CODE` via replaceState for persistence on refresh
4. Form collects: attendance (toggle), email, phone, dietary needs (pills), +1 name (solo invites only)
5. Hotel-eligible invites show Belmond booking section with images
6. Submit → botanical confetti if attending, "We'll miss you" if declining

## Admin

- `/admin` — password-protected (cookie-based, 30-day expiry, `ADMIN_PASSWORD` env var)
- Dashboard shows stats (attending/declined/pending/hotel bookings) + table of all invites
- Click any code → `/admin/rsvp/CODE` — edit all fields, mark hotel booking complete, set booking value
- "+ New Invite" button creates invites with auto-generated codes

## Design

- **Light mode only** — warm cream palette inspired by San Miguel de Allende (plaster walls, terracotta, sun-bleached stone)
- **Fonts**: Cormorant Garamond (serif, headings/toggle labels), Geist Sans (body), Geist Mono (codes)
- **Colors**: warm brown foreground (not black), muted terracotta primary, sand borders. All oklch in `globals.css`.
- **Design budget principle**: minimal visual treatments, spacing and typography do the hierarchy work
- **NoiseBackground**: animated warm gradients + noise texture on form inputs and toggle. Inset shadows for depth.
